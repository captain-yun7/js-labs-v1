HTML 링크

링크(hyperlink)는 웹의 핵심 요소로, 다른 페이지나 웹 리소스로 이동할 수 있게 합니다. HTML에서는 <a> 태그를 사용하여 링크를 생성합니다.

1. 기본 링크 생성

기본 링크는 다음과 같이 사용합니다:

Example 웹사이트

코드 예시:

<a href="https://www.example.com">Example 웹사이트</a>

href(필수): 링크가 가리키는 목적지 URL

2. 링크 대상 지정

target 속성을 사용하여 링크가 열리는 방식을 지정할 수 있습니다:

새 탭에서 열기

코드 예시:

<a href="https://www.example.com" target="_blank">새 탭에서 열기</a>

target 값:

  • _self: 현재 창에서 열기 (기본값)
  • _blank: 새 탭이나 창에서 열기
  • _parent: 부모 프레임에서 열기
  • _top: 전체 브라우저 창에서 열기

3. 링크 타이틀 추가

title 속성을 사용하여 링크에 대한 추가 정보를 제공할 수 있습니다:

Example 웹사이트 (마우스를 올려보세요)

코드 예시:

<a href="https://www.example.com" title="Example.com - 예제 웹사이트">Example 웹사이트</a>

4. 내부 페이지 링크 (같은 웹사이트 내)

같은 웹사이트 내의 다른 페이지로 연결할 때는 상대 경로를 사용할 수 있습니다:

이미지 페이지로 이동
HTML 기본 구조 페이지로 이동

코드 예시:

<!-- 같은 디렉토리 내의 파일 -->
<a href="images.html">이미지 페이지로 이동</a>

<!-- 상위 디렉토리의 파일 -->
<a href="../01-html-basics/basic-structure.html">HTML 기본 구조 페이지로 이동</a>

5. 페이지 내 특정 위치로 이동 (앵커 링크)

id 속성을 사용하여 페이지 내 특정 위치로 이동하는 앵커 링크를 생성할 수 있습니다:

섹션 1 섹션 2 섹션 3

섹션 1

이 섹션은 id="section1"을 가지고 있어 앵커 링크의 대상이 됩니다.

섹션 2

이 섹션은 id="section2"를 가지고 있어 앵커 링크의 대상이 됩니다.

섹션 3

이 섹션은 id="section3"을 가지고 있어 앵커 링크의 대상이 됩니다.

코드 예시:

<!-- 페이지 상단에 네비게이션 링크 -->
<a href="#section1">섹션 1</a>
<a href="#section2">섹션 2</a>

<!-- 페이지 내 해당 위치에 id 지정 -->
<div id="section1">
    <h3>섹션 1</h3>
    <p>섹션 1 내용...</p>
</div>

<div id="section2">
    <h3>섹션 2</h3>
    <p>섹션 2 내용...</p>
</div>

6. 다른 페이지의 특정 위치로 이동

다른 페이지의 특정 위치로 이동하려면 URL 뒤에 앵커(#)를 추가합니다:

이미지 페이지의 섹션 2로 이동

코드 예시:

<a href="images.html#section2">이미지 페이지의 섹션 2로 이동</a>

7. 이메일 링크

mailto: 프로토콜을 사용하여 이메일 클라이언트를 실행하는 링크를 만들 수 있습니다:

이메일 보내기

코드 예시:

<a href="mailto:example@example.com">이메일 보내기</a>

추가 옵션을 포함할 수도 있습니다:

<a href="mailto:example@example.com?subject=문의사항&body=안녕하세요,%20문의드립니다.">
    문의 이메일 보내기
</a>

8. 전화번호 링크

tel: 프로토콜을 사용하여 전화를 걸 수 있는 링크를 만들 수 있습니다 (모바일 기기에서 유용):

010-1234-5678로 전화하기

코드 예시:

<a href="tel:+8201012345678">010-1234-5678로 전화하기</a>

9. 다운로드 링크

download 속성을 사용하여 링크를 클릭했을 때 파일을 다운로드하게 할 수 있습니다:

텍스트 파일 다운로드

코드 예시:

<a href="파일경로.pdf" download="저장될_파일명.pdf">PDF 파일 다운로드</a>

10. 링크 스타일링

CSS를 사용하여 링크 모양을 변경할 수 있습니다:

일반 링크

기본 스타일 링크

버튼 형태 링크

버튼처럼 보이는 링크

네비게이션 링크

홈 소개 서비스 연락처

코드 예시:

<!-- CSS -->
a {
    color: #3498db;
    text-decoration: none;
}
a:hover {
    text-decoration: underline;
}
a:visited {
    color: #8e44ad;
}
.button-link {
    display: inline-block;
    padding: 10px 15px;
    background-color: #3498db;
    color: white;
    border-radius: 4px;
    text-decoration: none;
}
.button-link:hover {
    background-color: #2980b9;
}

<!-- HTML -->
<a href="#">기본 스타일 링크</a>
<a href="#" class="button-link">버튼처럼 보이는 링크</a>

참고: 링크는 웹 접근성을 고려하여 명확한 텍스트로 제공하는 것이 좋습니다. "여기를 클릭하세요" 같은 모호한 표현보다 "제품 설명서 다운로드"와 같이 구체적인 텍스트를 사용하세요.
