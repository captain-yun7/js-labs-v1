HTML 제목 태그

HTML 제목 태그는 웹 페이지의 구조와 계층을 나타내는 중요한 요소입니다. 검색 엔진은 제목 태그를 사용하여 페이지의 내용을 이해하고 페이지의 구조를 파악합니다.

제목 태그의 종류

HTML에는 6개의 제목 태그가 있습니다:

h1 태그: 가장 중요한 제목

h2 태그: 두 번째로 중요한 제목

h3 태그: 세 번째로 중요한 제목

h4 태그: 네 번째로 중요한 제목

h5 태그: 다섯 번째로 중요한 제목
h6 태그: 여섯 번째로 중요한 제목

제목 태그의 특징

  • 중요도에 따른 크기: h1이 가장 크고, h6이 가장 작습니다.
  • 굵은 글씨: 모든 제목 태그는 기본적으로 굵은 글씨로 표시됩니다.
  • 줄바꿈: 제목 태그는 블록 요소로, 앞뒤에 자동으로 줄바꿈이 삽입됩니다.
  • 여백: 브라우저는 제목 태그 주변에 기본 여백을 추가합니다.

제목 태그의 올바른 사용법

1. 페이지 구조 반영하기

제목 태그는 페이지의 구조와 계층을 반영해야 합니다. 논리적인 순서를 따라 h1부터 시작하여 하위 제목으로 진행해야 합니다.

올바른 예: h1 → h2 → h3 → h2 → h3

잘못된 예: h3 → h1 → h4 → h2

2. h1 태그는 페이지당 한 번만 사용하기

SEO(검색 엔진 최적화) 관점에서, 한 페이지에는 하나의 h1 태그만 사용하는 것이 좋습니다. h1 태그는 페이지의 주제를 나타내야 합니다.

3. 내용에 맞게 사용하기

제목 태그는 단순히 텍스트를 크게 보이게 하는 용도가 아니라, 콘텐츠의 구조와 계층을 나타내기 위해 사용해야 합니다.

제목 태그와 접근성

스크린 리더와 같은 보조 기술은 제목 태그를 사용하여 사용자가 페이지를 탐색할 수 있도록 도와줍니다. 논리적이고 잘 구조화된 제목 태그는 모든 사용자에게 더 나은 경험을 제공합니다.
