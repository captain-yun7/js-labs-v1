HTML 텍스트 서식

HTML은 웹 페이지의 내용과 구조를 정의합니다. 텍스트를 서식화하는 다양한 태그를 제공하여 내용의 의미와 중요성을 나타낼 수 있습니다.

제목 태그 (Headings)

제목 1 (h1)

제목 2 (h2)

제목 3 (h3)

제목 4 (h4)

제목 5 (h5)
제목 6 (h6)

코드 예시:

<h1>제목 1</h1>
<h2>제목 2</h2>
<h3>제목 3</h3>
<h4>제목 4</h4>
<h5>제목 5</h5>
<h6>제목 6</h6>

단락 (Paragraphs)

이것은 단락입니다. 단락은 본문의 기본 단위로, 관련된 내용을 함께 그룹화합니다.

새로운 단락은 새로운 줄에서 시작되며, 자동으로 위아래 여백이 적용됩니다.

코드 예시:

<p>이것은 단락입니다.</p>
<p>새로운 단락입니다.</p>

텍스트 서식화 태그

강조 텍스트 (strong) - 중요한 내용을 나타냅니다.

이탤릭체 텍스트 (em) - 강조하거나 대조하는 내용을 나타냅니다.

하이라이트 텍스트 (mark) - 주목해야 할 텍스트를 표시합니다.

작은 텍스트 (small) - 저작권이나 법적 텍스트와 같은 작은 글씨에 사용합니다.

삭제된 텍스트 (del) - 문서에서 삭제된 텍스트를 나타냅니다.

추가된 텍스트 (ins) - 문서에 추가된 텍스트를 나타냅니다.

아래 첨자 (sub) 및 위 첨자 (sup) - 수학 표기나 각주에 유용합니다.

코드 예시:

<strong>강조 텍스트</strong>
<em>이탤릭체 텍스트</em>
<mark>하이라이트 텍스트</mark>
<small>작은 텍스트</small>
<del>삭제된 텍스트</del>
<ins>추가된 텍스트</ins>
<sub>아래 첨자</sub> 및 <sup>위 첨자</sup>

인용 (Quotations)

웹 표준은 웹 사이트를 만들 때 기준이 되는 규칙으로, 접근성과 호환성을 보장하기 위해 중요합니다.

짧은 인용문은 이렇게 따옴표로 표시됩니다.

코드 예시:

<blockquote cite="출처 URL">
    긴 인용문...
</blockquote>

<p>짧은 인용문은 <q>이렇게 따옴표로 표시됩니다</q>.</p>

줄바꿈과 수평선

첫 번째 줄
두 번째 줄

수평선 아래 내용

코드 예시:

<p>첫 번째 줄<br>두 번째 줄</p>
<hr>
<p>수평선 아래 내용</p>

특수 문자

HTML에서는 특수 문자를 표시하기 위해 엔티티 코드를 사용합니다:

  • < - 작다 기호 (&lt;)
  • > - 크다 기호 (&gt;)
  • & - 앰퍼샌드 (&amp;)
  • " - 큰따옴표 (&quot;)
  • ' - 작은따옴표 (&apos;)
  • © - 저작권 기호 (&copy;)
  • ® - 등록 상표 (&reg;)

참고: HTML5에서는 시맨틱 태그를 사용하여 내용의 의미를 더 명확하게 표현할 수 있습니다.
