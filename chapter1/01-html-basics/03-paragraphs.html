줄바꿈

HTML에서 줄바꿈은 <br> 태그를 사용합니다. 이 태그는 닫는 태그가 없는 빈 요소(empty element)입니다.

참고: <br> 태그는 줄바꿈이 필요한 곳에만 사용해야 합니다. 단락 사이의 간격을 늘리기 위해 여러 개의 <br> 태그를 사용하는 것은 좋지 않은 방법입니다. 대신 CSS의 margin 속성을 사용하세요.

수평선

텍스트 서식

HTML은 텍스트를 다양한 방식으로 강조하기 위한 여러 태그를 제공합니다.

텍스트 서식 예제

강조(Strong) 텍스트는 <strong> 태그를 사용합니다.

굵은(Bold) 텍스트는 <b> 태그를 사용합니다.

기울임(Emphasized) 텍스트는 <em> 태그를 사용합니다.

이탤릭(Italic) 텍스트는 <i> 태그를 사용합니다.

밑줄(Underline) 텍스트는 <u> 태그를 사용합니다.

취소선(Strikethrough) 텍스트는 <s> 태그를 사용합니다.

하이라이트(Highlight) 텍스트는 <mark> 태그를 사용합니다.

아래 첨자Subscript는 <sub> 태그를 사용합니다.

위 첨자Superscript는 <sup> 태그를 사용합니다.

작은 텍스트는 <small> 태그를 사용합니다.

코드 텍스트는 <code> 태그를 사용합니다.

의미론적 태그 사용: HTML5에서는 의미론적(semantic) 태그 사용을 권장합니다. 예를 들어, 단순히 텍스트를 굵게 표시하기 위해서는 <b> 대신 <strong>을, 기울임꼴을 위해서는 <i> 대신 <em>을 사용하는 것이 좋습니다.

정의 목록

HTML에서는 용어와 그 정의를 나열하는 정의 목록을 제공합니다.

정의 목록 예제
