HTML 목록

HTML에서는 다양한 유형의 목록을 생성할 수 있습니다. 목록은 정보를 구조화하고 가독성을 높이는 데 유용합니다.

1. 순서 없는 목록 (Unordered List)

순서 없는 목록은 항목의 순서가 중요하지 않을 때 사용합니다:

  • 사과
  • 바나나
  • 오렌지
  • 포도

코드 예시:

<ul>
    <li>사과</li>
    <li>바나나</li>
    <li>오렌지</li>
    <li>포도</li>
</ul>

2. 순서 있는 목록 (Ordered List)

순서 있는 목록은 항목의 순서가 중요할 때 사용합니다:

  1. 물 끓이기
  2. 티백 넣기
  3. 3분 기다리기
  4. 티백 꺼내기

코드 예시:

<ol>
    <li>물 끓이기</li>
    <li>티백 넣기</li>
    <li>3분 기다리기</li>
    <li>티백 꺼내기</li>
</ol>

3. 정의 목록 (Definition List)

정의 목록은 용어와 그에 대한 설명을 나열할 때 사용합니다:

HTML
웹 페이지의 구조를 정의하는 마크업 언어
CSS
웹 페이지의 스타일과 레이아웃을 정의하는 언어
JavaScript
웹 페이지에 상호작용과 동적 기능을 추가하는 프로그래밍 언어

코드 예시:

<dl>
    <dt>HTML</dt>
    <dd>웹 페이지의 구조를 정의하는 마크업 언어</dd>
    
    <dt>CSS</dt>
    <dd>웹 페이지의 스타일과 레이아웃을 정의하는 언어</dd>
    
    <dt>JavaScript</dt>
    <dd>웹 페이지에 상호작용과 동적 기능을 추가하는 프로그래밍 언어</dd>
</dl>

4. 중첩 목록 (Nested Lists)

목록은 다른 목록 안에 중첩될 수 있습니다:

  • 과일
    • 사과
    • 바나나
    • 오렌지
  • 채소
    • 당근
    • 브로콜리
    • 시금치

코드 예시:

<ul>
    <li>과일
        <ul>
            <li>사과</li>
            <li>바나나</li>
            <li>오렌지</li>
        </ul>
    </li>
    <li>채소
        <ul>
            <li>당근</li>
            <li>브로콜리</li>
            <li>시금치</li>
        </ul>
    </li>
</ul>
