이벤트 객체 (Event Object)

이벤트 객체란?

이벤트 핸들러 함수는 이벤트가 발생했을 때 자동으로 이벤트 객체를 매개변수로 받습니다.

이 객체는 발생한 이벤트에 대한 다양한 정보와 이벤트를 제어하는 메서드를 제공합니다.

element.addEventListener('click', function(event) {
  // event는 이벤트 객체
  console.log(event.type);
});

4. 이벤트 제어 메서드

preventDefault()

이벤트의 기본 동작을 취소합니다.

이 링크를 클릭하면 이동하지 않습니다

stopPropagation()

이벤트 버블링을 중단합니다.
