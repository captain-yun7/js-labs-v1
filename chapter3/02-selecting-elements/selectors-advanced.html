CSS 선택자 기반 DOM 선택

querySelector와 querySelectorAll은 CSS 선택자를 사용하여 요소를 선택합니다:


// 첫 번째 매칭 요소
document.querySelector('.class-name')

// 모든 매칭 요소
document.querySelectorAll('#container .item')
