JavaScript 이벤트 유형

이벤트 유형 개요

JavaScript에서는 다양한 종류의 이벤트를 처리할 수 있습니다. 이벤트는 사용자 동작이나 시스템 이벤트에 의해 발생됩니다.

이 페이지에서는 주요 이벤트 유형을 카테고리별로 살펴보고 예제를 통해 실습해볼 수 있습니다.

1. 마우스 이벤트

사용자의 마우스 동작과 관련된 이벤트입니다.

이벤트 이름 설명
click 요소를 클릭했을 때
dblclick 요소를 더블 클릭했을 때
mousedown 마우스 버튼을 누를 때
mouseup 마우스 버튼을 뗄 때
mousemove 마우스를 움직일 때
mouseover 마우스가 요소 위로 들어올 때
mouseout 마우스가 요소에서 나갈 때
mouseenter 마우스가 요소에 들어올 때 (버블링 없음)
mouseleave 마우스가 요소에서 나갈 때 (버블링 없음)
contextmenu 마우스 오른쪽 버튼을 클릭했을 때

마우스 이벤트 데모

위 상자에 다양한 마우스 동작을 시도해보세요.

2. 키보드 이벤트

사용자의 키보드 동작과 관련된 이벤트입니다.

이벤트 이름 설명
keydown 키를 누를 때
keyup 키를 뗄 때
keypress 문자 키를 누를 때 (더 이상 권장되지 않음)

3. 폼 이벤트

HTML 폼과 관련된 이벤트입니다.

이벤트 이름 설명
submit 폼이 제출될 때
reset 폼이 초기화될 때
change 입력 요소의 값이 변경되고 포커스를 잃을 때
input 입력 요소의 값이 변경될 때 (실시간)
focus 요소가 포커스를 얻을 때
blur 요소가 포커스를 잃을 때

4. 문서/창 이벤트

문서나 창과 관련된 이벤트입니다.

이벤트 이름 설명
load 페이지와 모든 리소스가 로드 완료되었을 때
DOMContentLoaded HTML 문서가 파싱 완료되었을 때 (이미지, 스타일시트 등 기다리지 않음)
resize 창 크기가 변경될 때
scroll 페이지나 요소가 스크롤될 때
beforeunload 페이지를 떠나기 전
unload 페이지를 떠날 때

문서/창 이벤트 데모

창 크기를 조절하거나 페이지를 스크롤해보세요.

5. 드래그 앤 드롭 이벤트

HTML5의 드래그 앤 드롭 API와 관련된 이벤트입니다.

이벤트 이름 설명
dragstart 드래그 시작 시
drag 드래그 중
dragend 드래그 종료 시
dragenter 드래그한 요소가 드롭 대상에 들어갈 때
dragover 드래그한 요소가 드롭 대상 위에 있을 때
dragleave 드래그한 요소가 드롭 대상을 벗어날 때
drop 드래그한 요소를 드롭 대상에 놓을 때

6. 미디어 이벤트

오디오 및 비디오 요소와 관련된 이벤트입니다.

이벤트 이름 설명
play 재생이 시작될 때
pause 재생이 일시 중지될 때
ended 재생이 끝날 때
volumechange 볼륨이 변경될 때
timeupdate 현재 재생 시간이 변경될 때
