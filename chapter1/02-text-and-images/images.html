HTML 이미지

이미지는 웹 페이지를 더 흥미롭고 시각적으로 만들어줍니다. HTML에서는 <img> 태그를 사용하여 이미지를 삽입할 수 있습니다.

1. 기본 이미지 삽입

기본 이미지 태그는 다음과 같이 사용합니다:

코드 예시:

<img src="이미지_경로.jpg" alt="대체 텍스트">

src(필수): 이미지 파일의 경로(URL)

alt(필수): 이미지가 표시되지 않을 때 나타나는 대체 텍스트. 스크린 리더가 읽는 접근성을 위한 텍스트

2. 이미지 크기 지정

width와 height 속성으로 이미지 크기를 지정할 수 있습니다:

코드 예시:

<img src="이미지_경로.jpg" alt="대체 텍스트" width="400" height="200">

참고: 이미지의 원래 비율을 유지하기 위해 width만 지정하고 height는 auto로 설정하는 것이 좋습니다. 이는 CSS를 통해 적용할 수 있습니다.

3. 이미지 타이틀 추가

title 속성을 사용하여 마우스를 올렸을 때 표시되는 툴팁을 추가할 수 있습니다:

코드 예시:

<img src="이미지_경로.jpg" alt="대체 텍스트" title="이미지에 마우스를 올리면 이 텍스트가 보입니다">

4. 텍스트와 이미지 배치

이미지 왼쪽 배치

이 텍스트는 이미지 오른쪽에 배치됩니다. 이미지를 텍스트와 함께 배치할 때는 float 속성을 사용하는 것이 일반적입니다. 이 방법을 사용하면 텍스트가 이미지 주변을 감싸게 됩니다. 충분한 텍스트가 있으면 이미지 아래로 흐르게 됩니다.

이미지 오른쪽 배치

이 텍스트는 이미지 왼쪽에 배치됩니다. 오른쪽 배치도 float 속성을 사용하여 구현할 수 있습니다. 이렇게 하면 텍스트가 이미지의 반대편에 배치됩니다. 이 방식은 잡지 스타일의 레이아웃을 만들 때 유용합니다.

코드 예시:

<!-- CSS -->
.float-left {
    float: left;
    margin-right: 15px;
}
.float-right {
    float: right;
    margin-left: 15px;
}
.clearfix::after {
    content: "";
    clear: both;
    display: table;
}

<!-- HTML -->
<div class="clearfix">
    <img src="이미지.jpg" alt="이미지" class="float-left">
    <p>이미지 오른쪽에 배치되는 텍스트...</p>
</div>

5. 이미지 캡션 (Figure와 Figcaption)

figure와 figcaption 태그를 사용하여 이미지에 캡션을 추가할 수 있습니다:

코드 예시:

<figure>
    <img src="이미지_경로.jpg" alt="대체 텍스트">
    <figcaption>이미지에 대한 설명 텍스트</figcaption>
</figure>

6. 반응형 이미지

CSS를 사용하여 뷰포트 크기에 맞게 이미지 크기를 조정할 수 있습니다:

코드 예시:

<!-- CSS -->
.img-responsive {
    width: 100%;
    height: auto;
}

<!-- HTML -->
<img src="이미지_경로.jpg" alt="대체 텍스트" class="img-responsive">

7. 이미지 맵 (Image Map)

이미지 맵을 사용하면 이미지의 특정 영역을 클릭하여 다른 페이지로 이동할 수 있습니다:

코드 예시:

<img src="이미지_경로.jpg" alt="이미지 맵 예제" usemap="#example-map">

<map name="example-map">
    <area shape="rect" coords="0,0,200,200" href="page1.html" alt="왼쪽 영역">
    <area shape="rect" coords="201,0,400,200" href="page2.html" alt="오른쪽 영역">
</map>

8. 이미지 스타일링

CSS를 사용하여 이미지에 다양한 스타일을 적용할 수 있습니다:

썸네일 이미지

원형 이미지

코드 예시:

<!-- CSS -->
.img-thumbnail {
    border: 1px solid #ddd;
    border-radius: 4px;
    padding: 5px;
}
.img-circle {
    border-radius: 50%;
}

<!-- HTML -->
<img src="이미지_경로.jpg" alt="대체 텍스트" class="img-thumbnail">
<img src="이미지_경로.jpg" alt="대체 텍스트" class="img-circle">

참고: 이미지를 사용할 때는 항상 저작권에 주의하세요. 자신이 사용 권한이 있는 이미지만 사용하거나, 무료 이미지 사이트에서 제공하는 이미지를 활용하세요.
