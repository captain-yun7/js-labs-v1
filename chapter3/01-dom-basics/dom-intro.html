DOM(Document Object Model) 소개

DOM이란?

DOM은 Document Object Model의 약자로, HTML 문서를 객체로 표현한 것입니다.

DOM은 웹 페이지의 프로그래밍 인터페이스로, JavaScript를 통해 문서의 내용, 구조, 스타일을 변경할 수 있게 해줍니다.

DOM 트리

브라우저는 HTML 문서를 로드할 때, 문서의 내용을 트리 구조로 변환합니다.

이를 DOM 트리라고 하며, 각 HTML 요소는 트리의 노드가 됩니다.

예를 들어, 다음 HTML 구조:

<div>
  <h2>제목</h2>
  <p>단락 텍스트</p>
</div>

는 다음과 같은 DOM 트리로 변환됩니다:

document
  └── div
      ├── h2
      │   └── "제목" (텍스트 노드)
      └── p
          └── "단락 텍스트" (텍스트 노드)

JavaScript로 DOM 조작하기

JavaScript를 사용하면 DOM에 접근하여 요소를 추가, 수정, 삭제할 수 있습니다.

이것은 DOM 조작을 위한 데모 요소입니다.

DOM의 주요 객체

  • document - 웹 페이지 그 자체를 나타내는 객체
  • element - 모든 HTML 요소를 나타내는 객체
  • nodeList - 노드의 배열 형태
  • attribute - HTML 요소의 속성을 나타내는 객체
