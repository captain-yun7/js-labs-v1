HTML 기본 구조

HTML 문서의 구성 요소

HTML 문서는 다음과 같은 주요 구성 요소로 이루어져 있습니다:

  1. DOCTYPE 선언: <!DOCTYPE html> - 문서 유형을 정의합니다.
  2. html 요소: <html> - HTML 문서의 루트(최상위) 요소입니다.
  3. head 요소: <head> - 메타데이터와 문서 제목 등을 포함합니다.
  4. body 요소: <body> - 사용자에게 보여지는 내용을 포함합니다.

head 요소 내 중요 태그

  • <title>: 브라우저 탭에 표시되는 페이지 제목을 정의합니다.
  • <meta charset="UTF-8">: 문서의 문자 인코딩을 UTF-8로 설정합니다.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: 반응형 웹 디자인을 위한 설정입니다.
  • <link>: 외부 리소스(예: CSS 파일)를 연결합니다.
  • <script>: JavaScript 코드를 포함하거나 외부 JavaScript 파일을 연결합니다.

주석

HTML에서 주석은 <!-- --> 태그로 작성합니다. 주석은 브라우저에 표시되지 않으며, 코드에 설명을 추가하는 용도로 사용됩니다.

HTML 태그의 기본 형식

대부분의 HTML 태그는 다음과 같은 형식을 가집니다:

<태그명 속성명="속성값">내용</태그명>

예를 들어:

<a href="https://www.example.com">링크 텍스트</a>

빈 요소(Empty Elements)

내용이 없는 태그를 빈 요소라고 합니다. 이러한 태그는 일반적으로 다음과 같이 작성합니다:

<태그명 속성명="속성값">

또는 XML 스타일로:

<태그명 속성명="속성값" />

예를 들어:

<img src="image.jpg" alt="이미지 설명">
<br>
<hr>

이 페이지는 HTML 학습을 위한 예제입니다.
